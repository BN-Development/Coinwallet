All about address types

Bitcoin and Litecoin addresses are 26-35 characters long, consist of alphabetic and numeric characters, and either begin with “1”, “3”, or “bc1

1. P2PKH

Address starts with the number “1”

Example:
1BvBMSEYstWetqTFn5Au4m4GFg7xJaNVN2

P2PKH which begin with the number L
LLfeanwXA7LhR92Sst46uBYn94qsR2rJ2h.

2. P2SH

Address starts with the number “3”

Example:
3J98t1WpEZ73CNmQviecrnyiWrnqRhWNLy

P2SH type starting with the number M
MTfA1ofu5wp6Ss9yrXZXN2ikevdHT7ktqw.

3. Bech32

Address starts with “bc1”

Example:
bc1qar0srrr7xfkvy5l643lydnw9re59gtzzwf5mdq

Bech32 type starting with ltc
1ltc1q3qkpj5s4ru3cx9t7dt27pdfmz5aqy3wplamkns.
